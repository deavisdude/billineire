## Testing Checklist

When testing Roman structures in-game:

**Visual Verification**
- [ ] Structures at proper ground level (no giant pillars)
- [ ] Flat terracotta roofs (no peaked roofs)
- [ ] Column placement every 3 blocks on perimeter
- [ ] Glass pane windows on upper levels
- [ ] Mosaic floor patterns visible
- [ ] Courtyards open to sky (domus/villa)

**Functional Verification**
- [ ] Doorways clear and accessible
- [ ] Interior lighting adequate (no dark spots)
- [ ] Workstations properly placed (forge, anvil, crafting)
- [ ] Water features present (courtyards, baths)
- [ ] Campfires lit in bathhouse

**Placement Verification**
- [ ] Multiple structure types spawning
- [ ] No floating structures
- [ ] No embedded-in-hillside structures
- [ ] Structures rotated deterministically
- [ ] Natural terrain integration
